# Power Signature Blob


## Intructions

- Repository download:
```sh
git clone https://gitlab.com/fernando.delunogarcia/psb-knn-dataset.git
```

- Enviornment setup:
```sh
cd psb-knn-dataset

python3 -m venv env
. env/bin/activate
python -m pip install -r requirements.txt
```

- Running dashboard:
```sh
streamlit run dashboard.py
```

- Running notebook:
```sh
jupyter notebook notebook.ipynb
```
